# Project Files

This folder is populated by the Brief Architect skill (00) at the start of each project.

## Files generated:
- `project-brief.md` — Master document (user approves this)
- `design-direction.md` — Input for Design Director
- `page-plan.md` — Input for Page Planner
- `copy-brief.md` — Input for Copywriter
- `tech-spec.md` — Input for Component Architect
- `section-suggestions.md` — Recommended additions
- `brand-bible.md` — Client-facing deliverable

Do not manually create files here. Let Brief Architect generate them.
